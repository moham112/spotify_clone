## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/your-repo/spotify-clone.git
   ```
2. Navigate to the project directory:
   ```sh
   cd spotify-clone
   ```
3. Install dependencies:
   ```sh
   flutter pub get
   ```
4. Run the app:
   ```sh
   flutter run
   ```
